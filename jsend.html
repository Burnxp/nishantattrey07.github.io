What is JAVASCRIPT?


    JAVASCRIPT is a programming language that is used to create dynamic and interactive content on websites. It allows developers to implement complex features on web pages, such as animated graphics, interactive forms, and real-time updates.

Variables in JS

Syntax: 

            var x = 5;
            let y = 10;
            const z = 15;
        
    var = variable that can be redeclared and updated
    let = variable that can be updated but not redeclared   
    const = variable that cannot be updated or redeclared

Functions in JS

Syntax: 

            function myFunction(p1, p2) {
                return p1 * p2;   
              }
        
The function returns the product of p1 and p2
